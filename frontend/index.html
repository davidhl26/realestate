    🔭 Zillow Watch

    Set your criteria once — the watch re-runs the search, remembers what it has seen, and flags 🆕 new listings, 📉 price drops, and ❌ listings that are gone.

    The server re-runs each watch on its own at the chosen frequency (even with the app closed): new Zillow listings arrive already analyzed (estimated ARV/rehab/profit) in the feed. 1 refresh = 1 AI search ≈ $0.20-0.40 → hourly ≈ $5-9/day per watch, 1×/day ≈ $0.30/day.

    Import auctions

    Three ways to add auctions — bulk list, single detail URL, or fully manual.

    Paste a DAYLIST URL — we'll auto-drill into each property's detail page. Slow (1-3 min for 50+ items).

    Paste a SINGLE auction detail URL (with AID=) — scrapes just that one. Fast (~5-10 sec).

    Type everything manually. Use this when scraping fails or you have the data already.

    🎯 Deal Radar

    A find must pass all criteria: margin ≥ your target, profit ≥ this amount, the 70% rule, and low risk (grade A/B).

    Zillow blocks direct scraping (anti-bot). With a ScraperAPI key (free, 1000 req/month), every Zillow URL loads all photos automatically. 2-min signup: scraperapi.com/signup → copy your key from the dashboard.

    When a site (Zillow…) blocks scraping, we attach a Street View photo of the house from the address. Free key (generous tier): console.cloud.google.com — enable Street View Static API.
